# Praca-dyplomowa
Aplikacja internetowa do rezerwacji miejsc w restauracjach (JAVA, Spring Boot, JavaScript, React, MySQL)


Celem niniejszej pracy jest wykonanie projektu aplikacji webowej, 
za pomocą znanych na rynku i nowoczesnych technologii programowania,
przeznaczonej dla większości przeglądarek internetowych. 
Do stworzenia aplikacji wykorzystamy integrację technologii Java, Spring Boot, React oraz bazy danych MySQL.
Każdy zainteresowany skorzystaniem z aplikacji może utworzyć konto, co uprawni go do wykonywania rezerwacji w istniejących lokalach w bazie danych. 
Po zalogowaniu użytkownikowi wyświetla się widok jego konta, z którego może przejść do edycji danych, listy rezerwacji oraz dostępnych restauracji w jego mieście. 
Kolejną funkcją dostępną dla użytkowników po zalogowaniu jest możliwość dodania swojej restauracji do systemu. 
Takiemu użytkownikowi pojawi się możliwość zarządzania rezerwacjami, zmianą danych oraz ich edycją.
